# Owen Pallett Lyrics Bot

A bot to tweet random lines from a corpus made up of the complete lyrics of Owen Pallett (formerly Final Fantasy). Nothing clever happening with APIs or anything, just text copy-pasted from [Alpentine](http://alpentine.com). 
Based on [Mary Dickson's tutorial](http://marydickson.com/build-a-twitter-bot-with-python/) but altered to remove sleep time and add a random number generator so it'll work with Heroku's scheduler without tweeting too frequently.
